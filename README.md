# Système Monétaire Bimétallique CBU-X

> **Une architecture entropique pour la souveraineté économique méditerranéenne**

## 🎯 Vue d'ensemble

Le **CBU-X** (Currency Basket Unit — eXchange) est un cadre théorique et computationnel articulant trois briques mathématiques :

1. **Flux entropique** `Φ(t)` — Mesure la dissipation de valeur dans les échanges commerciaux
2. **Transport optimal** (Sinkhorn) — Optimise la logistique inter-portuaire
3. **Waqf numérique** — Redistribue sans intérêt via contrat intelligent Solidity
